## Getting Started

### Install Yeoman

```bash
$ npm install -g yo
```
### Install Generator

To install generator-static from npm, run:

```bash
$ npm install -g generator-clock-frontend
```

### Create Project Directory

```bash
$ mkdir myproject
$ cd myproject
```

### Initiate Generator

```bash
$ yo clock-frontend
```
Then follow the instructions.
